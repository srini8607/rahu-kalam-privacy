# Privacy Policy for Rahu Kalam Alexa Skill

This Alexa skill does not collect, store, or share any personal information.

The skill may request access to device location (country and postal code) only to determine the correct local day for calculating Rahu Kalam time. This information is used solely at runtime and is not stored, logged, or shared with any third parties.

No user data is retained outside of the Alexa service.

If you have any questions about this privacy policy, please contact the skill developer.
